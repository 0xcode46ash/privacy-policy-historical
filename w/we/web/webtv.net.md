# WebTV Privacy Policy

**WebTV Networks, Inc.**  
**Web Site Statement of Privacy**

WebTV Networks, Inc., a wholly owned subsidiary of Microsoft Corporation hosts the Microsoft® WebTV Network™, which enables subscribers to access the Internet utilizing a WebTV- based Internet unit, a television and a telephone line. This Privacy Statement applies to the WebTV Networks, Inc. Corporate Web site you are currently viewing and informs you what information is collected on this Web site. WebTV Networks, Inc. has a separate privacy policy that applies to its subscribers to the WebTV Network Service which is accessible from this Web site and can also be found by WebTV Network subscribers within the "Using Web TV" area of the WebTV Network Service. 

WebTV Networks, Inc. is a licensee of and certified by TRUSTe, an independent, non-profit initiative that exists to help people such as yourself feel confident about using the Internet for your communication, shopping, research, and living. TRUSTe aims to build this confidence by promoting the principles of disclosure and fair information practices among the Web sites that participate in the program. What this means to you is, we are committed to protecting your privacy and developing technology that gives you the most powerful, safe, online experience that you can get anywhere.

**Privacy Statement**

This WebTV Networks, Inc. Corporate Web site does not request any personal information or information that personally identifies you individually or allows you to be personally contacted. Since we do not collect any personal information on this Web site, we do not share any personal information with any third parties nor do we use any personal information for any purposes. 

This Privacy Statement applies only to the WebTV Networks, Inc. Corporate Web site and not to other companies or organizations to which we may provide links from this Web site. If we make any changes to this Privacy Statement, we will post changes here so we can keep you fully informed. If at any time, you believe that the WebTV Networks, Inc. Corporate Web site has not adhered to this principle, please notify WebTV Networks, Inc. by email at [WebTVPrivacy@corp.webtv.net](mailto:WebTVPrivacy@corp.webtv.net) and we will use all commercially reasonable efforts to promptly determine and correct the problem.
