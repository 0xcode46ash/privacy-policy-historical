| The Mead Johnson Nutritionals web site is maintained by Mead Johnson Nutritionals offices at 2400 W. Lloyd Expressway, Evansville, IN 47721.  We can be reached by e-mail at [enfamilresourcecenter@enfamil.com](mailto:enfamilresourcecenter@enfamil.com)

|  |  |  |  |  |   
Use of Registration Information  
|  |  |  |  |  |   
  | 1. | We will collect and store personal information about you and the individuals in your household if you provide this information to our company.   
|  |  |  |  |  |   
  | 2. | If you have already submitted personal information and would like us to remove it from our records, please contact us at the e-mail address above. We will make reasonable efforts to delete your information within 48 hours of your request.  
|  |  |  |  |  |   
  |  |  |  |  |  |   
3. | We will not sell (or otherwise provide) your name, address, or phone number to a third party, with the following possible exceptions:  
|  |  |  |   
| a) |  | We may share the information you provide us with other Bristol-Myers Squibb related companies (this may include Bristol-Myers Products, and Bristol-Myers Squibb Worldwide Medicines Group).  
|  |  |  |   
| b) |  | We may send your name and address to subcontractors to fulfill your request for a coupon, product, product information, product samples, or offers made in connection with forms.  
|  |  |  |  |  |   
  | 4. |  | Personal information provided to us will enable us to improve our understanding of the nutrition needs and concerns of you and your family.  
|  |  |  |  |  |   
  | 5. |  |  | We may use the information to contact you via e-mail, phone, mail or events in your local area.  
|  |  |  |  |  |   
  | 6. |  | We may collect information about you that is related to your Internet usage, such as the type of Internet Browser you are using, the type of operating system used on your computer, the website from which you came prior to visiting our site, and the domain name of your internet connection.  
|  |  |  |  |  |   
  | 7. |  | **Use of Cookies** \- We may store some information on your computer to improve the quality of your visit to our site. In some cases, this information will help us to tailor repeat visits to the site.  With most Internet Browsers, you can erase this information, referred to as "cookies" from your computer's hard drive, block all cookies, or even receive a warning before a cookie is placed on your system.  Refer to your browser instructions or help button to learn more about cookies.  
|  |  |  |  |  |   
  | 8. |  | Mead Johnson does not intend to collect any personal information from children under thirteen years old.  Mead Johnson will specifically instruct children not to submit such information on our website.  If a child has provided us with any information, the parent or guardian of that child should contact us at the e-mail address above if they want this information deleted from our files.
