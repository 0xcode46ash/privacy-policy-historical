# PRIVACY STATEMENT

This privacy statement sets out Ecobank’s current policy and demonstrates its commitment to your financial privacy. 

When using the public areas of this website, user is not required to provide Ecobank with any personal information if he/he does not wish to do so and Ecobank does not monitor or collect any personally identifiable information from the user on his/her use of the public areas of this website. Ecobank may track the number of users who visit this website, but this tracking will not identify you. 

If user registers to use the password protected areas of this website, he/she will be asked to provide certain data about himself/herself which will be handled in accordance with applicable laws. Ecobank maintains strict security standards and procedures with a view to preventing unauthorized access to the user’s data. Ecobank uses available leading technologies such as data encryption, firewalls, and server authentication to protect the security of users’data.

A cookie marks and identifies visitors on an Internet website. If user does not wish to receive cookies,  the user can set his/her browser to alert him/her of a cookie being offered, or to reject all cookies. 

Users may ask any questions about privacy issues by e-mailing us at  [ infoweb@ecobank.com](mailto:infoweb@ecobank.com)
