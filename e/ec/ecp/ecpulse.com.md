|    |   
|  The privacy policy published on this page will be applied to all users of the Company's website. If you do not accept this policy, we kindly ask you not to use this website in order to avoid any conflict. 

As a user/visitor, when you enter our website it implicitly means that you have read and accepted our privacy policy published on this page.

|   
|    |   
|  Non-Personal Information |   
|    |   
|  Economic Pulse guarantees the privacy of all visitors of the website. Every once in a while, the company may provide data to a third party, but only for general statistics regarding the number of users of the website and for marketing campaigns. These statistics will not include any personal information about the website users. |   
|    |   
|  Personal Information |   
|    |   
|  Users/visitors have to provide personal information like Name, Postal Address, Phone number, E-mail for us to answer all their questions and inquiries. This information will only be used to contact users/visitors of the website in order to answer their questions and inquiries, as well as to inform them about the services and the instruments that the company will offer in the future. |   
|    |   
|  Privacy |   
|    |   
|  Economic Pulse would like to draw your attention on the fact that we use the best technological methods and means to maintain the privacy of the personal information of the website users. However, there is no comprehensive protection system that offers full protection for personal information, thus making it vulnerable to hacking. |   
|    |   
|  Underage Visitors |   
|    |   
|  People that are below 18 years are not allowed to send any information or data to the company. Nor are they allowed to do any legal action on the website of Economic Pulse. |
