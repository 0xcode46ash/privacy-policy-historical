**Data Collection**

P3P policies declare the data they collect in groups (also referred to as "statements"). This policy contains 2 data groups. The data practices of each group will be explained seperately. 

**Group "Online User Information"**

We collect the following information: 

  * Full IP address 
  * HTTP cookies 
  * Transaction history 
  * User's interaction with a page or resource 
  * Unique identifiers 

This data will be used for the following purposes: 
  * Web site and system administration. The user is allowed to [opt-out](https://web.archive.org/web/20030409094319id_/http%3A//www.exclusivebooks.com/privacy/opt_out.asp?Tag=&CID=) of this usage. 
  * Research and development. The user is allowed to [opt-out](https://web.archive.org/web/20030409094319id_/http%3A//www.exclusivebooks.com/privacy/opt_out.asp?Tag=&CID=) of this usage. 
  * One-time tailoring. The user is allowed to [opt-out](https://web.archive.org/web/20030409094319id_/http%3A//www.exclusivebooks.com/privacy/opt_out.asp?Tag=&CID=) of this usage. 
  * User analysis. The user is allowed to [opt-out](https://web.archive.org/web/20030409094319id_/http%3A//www.exclusivebooks.com/privacy/opt_out.asp?Tag=&CID=) of this usage. 
  * User profiling and decision-making. The user is allowed to [opt-out](https://web.archive.org/web/20030409094319id_/http%3A//www.exclusivebooks.com/privacy/opt_out.asp?Tag=&CID=) of this usage. 

This data will be used by ourselves and our agents.  
The following explanation is provided for why this data is collected:  
Your IP Address is automatically collected and used for the following purposes: 

  * Quality control - to evaluate the technical performance of our web site. 
  * Diagnosing problems with our server. 
  * Administration of our web site. 
  * Gather broad demographic information. 
  * Assessing the utility and popularity of our product features and site areas. 

Exclusivebooks.com uses http cookies on its web site. These cookies only contain your name, e-mail address and a unique session identity number. The cookie is used to welcome you to our web site and to keep track of your shopping basket. Please note that disabling the cookies function on your web browser will prevent you from making any purchases on our web site.

**Cookies**

Cookies are a technology which can be used to provide you with tailored information from a Web site. A cookie is an element of data that a Web site can send to your browser, which may then store it on your system. You can set your browser to notify you when you receive a cookie, giving you the chance to decide whether to accept it.

Our site makes use of cookies. Cookies are used for the following purposes: 

  * Site administration 
  * User targeting 
  * User analysis 
  * Individual-based decision-making 
  * Research and development 
  * Keeping track of a user's shopping basket (Disabling cookies will make it inconvenient for you to place orders on our site)
