# Privacy Policy - nonoba.com

_Nonoba is run and owned by Odpy ApS. For more information see section "About Nonoba" in our[terms of service](https://web.archive.org/web/20120506001900id_/http%3A//nonoba.com/about/termsofservice)_

  1. ODPY respects users’ personal data and will comply with applicable data protection regulations. 

  2. ODPY collects the following information about the user in order to improve the service:

    * User’s name (and/or login name), age, gender, region, and country.
    * Information which other users publish on and attach to the user’s profile. 
    * User’s evaluations on games and other users.
    * Other information User may decide to post on the Website. 
  3. ODPY stores a cookie on the user’s computer to allow the user not to have to re-enter the user’s password during visits to the website. For the best experience the user is encouraged to enable/allow cookies. However, if cookies are deleted or if the browser is configured to reject cookies, the website can still be accessed by the user.

  4. ODPY may compile users’ personal data to produce statistics about the use of services and games. Such statistical information includes, but is not limited to, information showing the relative popularity of games and tools.

  5. ODPY shall not sell or share personal data with third parties. In the event that information is shared with third parties such information will not be personally identifiable information.

  6. By accepting the Terms of Use the user explicitly consents to use of personal data in accordance with this privacy Policy.
