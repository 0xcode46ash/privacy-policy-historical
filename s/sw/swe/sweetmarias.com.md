# Sweet Maria's Privacy Policy

WE DO NOT SELL, TRADE OR SHARE CUSTOMER INFORMATION OR MAIL LIST SUBSCRIBER INFORMATION WITH ANYONE ...EVER! 

WE DO NOT USE COOKIES, OR OTHER INFORMATION-GATHERING HTML CODE ON OUR WEB PAGES.

WHEN USING OUR SECURE ORDER FORM, ONLY THE INFORMATION YOU ENTER IS RETURNED TO US. IT IS ONLY RETURNED TO US, NOBODY ELSE.

(IT DOES TELL ME WHAT WEB BROWSER PROGRAM (I.E. ...NETSCAPE) YOU ARE USING WHEN YOU SUBMIT THE FORM, BUT ONLY BECAUSE I CAN'T FIGURE OUT HOW TO DELETE THIS PART OF THE CODE.)

WE DO NOT COLLECT TRAFFIC DATA EXCEPT THE NUMBER OF HITS TO OUR MAIN PAGE.

Please email Tom at sweetmarias@columbus.rr.com if you have any questions or concerns.
