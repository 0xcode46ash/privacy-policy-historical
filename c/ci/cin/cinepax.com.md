# Privacy Policy

A user may be required to give personal details for using some of the features of this website. Cinepax Limited does not sell this personal information to any commercial establishments nor does it spam users with unsolicited information. If a user has received any unsolicited information from Cinepax Limited then such user shall immediately inform Cinepax Limited of such incident.

However any such incident will not make Cinepax Limited or its representatives liable for any direct, indirect, special, incidental or consequential damages or loss (including damages for loss of business, loss of profits, or the like).
