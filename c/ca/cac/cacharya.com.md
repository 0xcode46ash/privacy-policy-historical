# Privacy Policy - Cacharya.com

[](https://web.archive.org/)

[ ](http://play.google.com/store/apps/details?id=com.cacharya)

We have created this privacy statement in order to demonstrate our firm commitment to privacy. The following discloses our information gathering and dissemination practices. 

  * **Server Logged Information**  
Whenever a user views a web site, the Web server automatically logs certain information about the visitor. This information includes the user's IP address, browser, and where the user came from. This is standard information collected by most Web servers. 

  * **Data Submission**  
Any data that passes through our system may be logged by us. 

  * **Cookies**  
Some scripts on our service or third party service we use, like Google Analytics, may utilize cookies to store information that is voluntarily submitted.
