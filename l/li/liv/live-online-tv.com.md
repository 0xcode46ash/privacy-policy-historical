**Cookies**   
We use cookies on this website. A cookie is a text file sent by a web server to a web browser, and stored by the browser. The text file is then sent back to the server each time the browser requests a page from the server. This enables the web server to identify and track the web browser.   
We may send a cookie which may be stored on by your browser on your computer’s hard drive.   
However they are not used to extract any information from you. We do not track or monitor these "cookies".   
Our advertisers may also send you cookies. Any information collected from users by advertisers using cookies within the Live-Online-TV.com is not subject to the Live-Online-TV.com privacy policy, but rather the privacy policy of the individual advertiser.
