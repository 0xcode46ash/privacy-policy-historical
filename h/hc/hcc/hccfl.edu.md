# Privacy - Hillsborough Community College - HCC

**PURPOSE  
**This procedure establishes guidelines concerning Hillsborough Community College’s (HCC) use of personal information collected on the HCC website and outlines HCC’s privacy statement concerning use of this information

**PROCEDURE

1\. PRIVACY STATEMENT

**HCC does not collect personal information when someone visits our website unless the user affirmatively chooses to make such information available to the College. Any information provided is not shared with anyone other than authorized HCC staff and  
faculty unless the College is required to do so by state or federal law. This privacy statement applies only to the information collected by web addresses in the hccfl.edu domain, hereafter referred to as HCC.

**2\. LOG FILES  
**HCC’s website does gather certain information automatically and store it in log files.This information may include internet protocol (IP) addresses, browser type, referring/exit pages, operating system, date/time stamp, and clickstream data. This information will be used to analyze trends in order to improve the content of the Web site, administer the site, and to track users" movements around the site, This information does not identify individual users, therefore personal e-mail addresses or other personal information are not captured.

**3\. COOKIES  
**A cookie is a small text file that is stored on a user’s computer for record-keeping purposes. HCC uses session ID cookies to make it easier to navigate the site and to verify proper access to secure areas once the user has successfully logged into those  
’areas. A session ID cookie expires and is deleted when the user closes the browser. Users should log off public computers browsers when their sessions are completed. Rejecting cookies still allows the user access to HCC’s site, but the ability to access  
some areas will be limited or denied. HCC requires the use of cookies to access all computing resources. 

**4\. PROTECTING YOUR PERSONAL INFORMATION  
**The HCC website uses industry-standard Secure Sockets Layer (SSL) encryption on all web pages where personal information is required. When entering personal information, the College strongly recommends the use of an SSL-enabled web browser.  
If one uses this Web site to make a financial transaction, one must be sure that the transaction is secure as indicated by the icon in the shape of a lock appearing on your screen. This will protect the confidentiality of personal and credit card information while  
it is transmitted across the Internet.

**5\. LINKS TO OTHER SITES  
**The HCC web site contains links to other sites that are not owned or controlled by HCC. HCC is not responsible for the privacy practices of such other sites. Users should read the privacy statements of each and every web site that collects personal identifiable  
information.

**6\. CHANGES TO THIS PRIVACY STATEMENT  
**If HCC decides to change the privacy policy, the College will post those changes to this privacy statement, on the home page as well as other places deemed appropriate. HCC reserves the right to modify this privacy statement at any time.
