# Contact Us - HTML Code Tutorial

[ ](https://web.archive.org/)|   
---|---  
We are committed to protecting your privacy at HTML Code Tutorial. We will not collect any personal information from you that you do not volunteer, and we are the sole owner of all information collected on this site. We do not sell, share, or rent this information to others. 

**Cookies**

Like many other websites, HTML Code Tutorial uses cookies to identify members so they don't have to re-enter a user id and password when they, for example, wish to post to a discussion. 

**Third parties**

HTML Code Tutorial does not provide third parties with aggregate statistics about our customers, traffic patterns, or related site information. 

**Obtaining your consent**

By using our Website, you give HTML Code Tutorial permission to collect information about the pages served to you as an anonymous user for the purpose of calculating aggregate site statistics. If you register at HTML Code Tutorial, you authorize HTML Code Tutorial to use information you provide to personalize the information we deliver to you and to use your demographic information when calculating aggregate customer data. HTML Code Tutorial may occasionally amend this policy. We suggest that you bookmark this page and check to see if it has changed from time to time. Questions? If you wish to change any information you have given us, please contact us at [learn@htmlcodetutorial.com](mailto:learn@htmlcodetutorial.com) .
