# Privacy Statement

**Overclockers.com Forums** has created this privacy statement in order to demonstrate our firm commitment to privacy. The following discloses the information gathering and dissemination practices for this Web site: [http://forum.oc-forums.com ](http://forum.oc-forums.com/)

**To contact us, regarding any privacy concerns, you can reach us at:**

Naperville, IL 60540

630-778-1874  


**We provide the following level of access to users information collected on this site:**  
We give access to Screen Name and Email Address if you elect to have Email Address public.

**We collect the following types of information:**  


  * User Information 
    * User Name 
    * User Birth Date 
    * User Email Address 
    * Volunteered User Profile Information 
  * Dynamic Information 
    * Http Cookies 
    * Http Information 
    * High Level Information Categories 
    * Online Contact Information 
    * Computer Information (Browser and OS Information) 
    * State Management Mechanisms 



**We use this information for:**  
Web Site and System Administration  
Completion and Support of Current Activity

**This data may be shared with:**  
Ourselves, Admins and Moderators

**We retain this information:**  
Data retained as required by law or liability under applicable law.
