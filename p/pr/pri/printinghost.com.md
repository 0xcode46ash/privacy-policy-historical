<span 9pt;="" verdana="" ,="" sans-serif="" ;="" color:="" black;="">In order to use My PrintingHost, a user mustfirst complete the registration form.  **<span 9pt;="" verdana="" ,="" sans-serif="" ;="" color:="" #3f3f3f;=""> Order**

<span 9pt;="" verdana="" ,="" sans-serif="" ;="" color:="" black;="">We request information from the user on ourorder form.  **<span 9pt;="" verdana="" ,="" sans-serif="" ;="" color:="" #3f3f3f;=""> Cookies**

<span 9pt;="" verdana="" ,="" sans-serif="" ;="" color:="" black;="">A cookie is a piece of data stored on the user’scomputer tied to information about the user. We use persistent cookies. Apersistent cookie is a small text file stored on the user’s hard drive for anextended period of time. Persistent cookies can be removed by followingInternet browser help file directions.

Cookies are used by both the My PrintingHost and the PrintingHost order sectionin order to uniquely identify users, associate user files with orders, and toenable the shopping basket. Our site cannot be used with cookies turned off.

This data is not linked to personally identifiable information.

 **<span 9pt;="" verdana="" ,="" sans-serif="" ;="" color:="" #3f3f3f;=""> Supplementation ofInformation **
